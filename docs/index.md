## Custom Project

### Dataset

The dataset used was sales.csv, which contains sales transaction records for
online learning products. Fields include order ID, date and time, region,
currency, product ID, unit price, quantity, customer information, payment
method, referral source, discount code, and customer notes. I used the original
sales dataset without modifications.

### Kafka Messages

My producer sent sales transaction messages to the
streaming-06-scenarios-kjleopold topic. Each message contained a sales record in
JSON format and used the region as the message key. I did not modify the
original message fields.

### Consumer Processing

My consumer received sales transaction messages from Kafka, validated the
records, calculated derived fields, stored valid records in DuckDB, and wrote
them to a CSV file. It consumed 50 messages and logged sales totals, summary
statistics, and DuckDB analytics. The consumer also calculated a new
customer_value field that classified transactions as low, medium, or high value
based on the total sale amount.

### Experiments

For Phase 4, I added a derived field called customer_value that classifies sales
as low, medium, or high based on the total sale amount.

For Phase 5, I extended the analytics pipeline by storing the new field in
DuckDB and creating a summary query that counts transactions by customer value
category.

### Results

The producer and consumer ran successfully. The consumer processed 50 messages,
stored the records in DuckDB, and generated summaries by region and customer
value category. The final results showed 42 low-value transactions, 6
medium-value transactions, and 2 high-value transactions based on the
customer_value classification.

### Interpretation

This project showed how streaming data can be processed, stored, and analyzed
using Kafka and DuckDB. The customer value classification made it easier to see
spending patterns in the data. Most transactions were low-value sales, while
only a few were medium- or high-value purchases. A business could use this
information to better understand customer spending habits and identify
opportunities to increase higher-value sales.
